# 2-TryHackMe-Labs

This folder contains concise reports of Blue Team / SOC-focused TryHackMe learning paths and challenges.

## How the reports are structured
Each report uses:
- Objective
- Steps performed
- Tools used
- Key findings
- Mapping to MITRE ATT&CK (if applicable)
- Lessons learned
